## Encrypted testimonials

Some of the testimonials are encrypted because they are private or for members of a review site only. The source for these files are encrypted using [Ansible Vault](https://docs.ansible.com/projects/ansible/latest/vault_guide/index.html), a simple string or file encryption tool.
